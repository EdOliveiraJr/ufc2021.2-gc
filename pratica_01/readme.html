Pratica 01 - Git Básico


Aguns comandos básicos do Git Bash:

1. "pdw": mostra o endereço do diretório que você se encontra
2. "cd nome_diretorio": acessa o diretório indicado
3. "ls": lista o conteúdo que consta no diretório.
4. "mkdir nome_novo_diretorio": cria novo diretório dentro do que você se encontra
5. 'vim nome_arquivo.extencao": abre um visualizador/editor de arquivo, se o aqruivo ainda não exite ele cria, se não abre o existente
6. "clear": limpa a console.
7. "git mv nome.extencao_velho nome.extencao_novo": renomear o arquivo

Alguns comando do vim

1. tecla "i": entra no modo de inserção/edição de texto.
2. tecla "esc": sai do modo de inserção, abrindo para digitar comandos.
3. ":q" : sair do vim.
4. ":wq" : salvar o documento e sair do vim.
